# Weekly Product Pulse: User Friction Points (2025-11-10 to 2025-11-16)
_Week: 2025-11-10 - 2025-11-16_

This week's pulse highlights significant user concerns across key areas. Financial transparency and system reliability emerged as critical issues, with users frequently reporting high charges and frustrating glitches. Additionally, several UI/UX enhancements and feature consistency requests were noted, impacting overall user experience and trust.

## Themes
- **Payments/Statements** - Users express strong negative sentiment regarding perceived high and hidden charges, impacting trust and transaction costs.
- **Slow, Glitches** - Critical issues with system reliability and order execution continue to cause frustration, potentially leading to user financial losses.
- **UI/UX** - Users provide clear feedback for specific functionality expansions and consistency improvements, particularly concerning trading interfaces and content accessibility.

## Quotes
- "one of the worst app having highest brokerage charges (Payments/Statements)"
- "sometimes gets glitches not common glitch but the bigger one (Slow, Glitches)"
- "Please make basket order available for both intraday and delivery. (UI/UX)"

## Actions
- Initiate a comprehensive review of all transaction fees to ensure transparency and competitiveness, addressing user concerns about hidden charges.
- Conduct an urgent investigation into critical order execution delays and 'exit all position' failures to restore system reliability.
- Standardize the application of Stop Loss and Take Profit features across all charting interfaces for a consistent user experience.
